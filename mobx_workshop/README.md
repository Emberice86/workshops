## React native exmple with MobX and Redux

### Technologies used:
- [ReactNative](https://facebook.github.io/react-native/)
- [TypeScript](https://www.typescriptlang.org/)
- [Redux](https://github.com/reactjs/redux)
- [MobX](https://github.com/mobxjs/mobx)

### To Start project:
    - Clone this repo
    - In mobx_workshop run `yarn`
    - Install typescript if it not install yet `npm i -g typescript`
    - Compile typescript files and watch for changes by running `tsc -w`
      @Note: for now there is be an error with types in react-redux, it will be fix later
    - Run `react-native run-ios` to launch project in ios simulator
